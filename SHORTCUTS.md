| Shortcut | Tool |
|----------|------|

## Editing Operations

| Shortcut | Action |
|----------|--------|
| `Ctrl+Z` | Undo (50 levels) |
| `Ctrl+Y` | Redo |
| `Ctrl+Shift+Z` | Redo (alternative) |
| `Ctrl+C` | Copy selected shapes |
| `Ctrl+V` | Paste copied shapes |
| `Ctrl+D` | Duplicate selected |
| `Ctrl+A` | Select all shapes |
| `Delete` | Delete selected shapes |
| `Backspace` | Delete selected shapes |

## Property Editing (When Shape Selected)

In the right panel, you can edit:
- **X/Y Position** - Type numbers or drag
- **Width/Height** - Resize dimensions
- **Rotation** - Angle in degrees (0-360)
- **Fill Color** - Click color picker
- **Stroke Color** - Click color picker
- **Stroke Width** - Border thickness
- **Opacity** - Transparency 0-100%
- **Text Content** - Edit text directly
- **Lock/Unlock** - Prevent accidental changes

## Tips for Speed

### Fastest Workflow
1. **Select Tool**: Press `V`
2. **Rectangle**: Press `R`, drag on canvas
3. **Style**: Edit in right panel
4. **Copy**: `Ctrl+C`
5. **Paste**: `Ctrl+V`
6. **Save**: `Ctrl+S`
7. **Undo**: `Ctrl+Z`

### Pro Shortcuts
- Use `Shift+Click` for multi-selection
- Use `Ctrl+D` to quickly duplicate
- Use `Ctrl+A` then align tools for perfect layout
- Use grid + snap for precise positioning
- Use `Ctrl+Z` liberally - 50 undo levels!

### Canvas Navigation
- **Zoom**: Scroll wheel (with Ctrl modifier)
- **Pan**: Alt + Click + Drag
- **Quick Zoom**: Ctrl++ or Ctrl+-
- **Fit to View**: Click maximize icon
